Requerimientos Mínimos.

  • 64 MB en RAM. 
  • Java 2
  • Java API 3D
  • Procesador mínimo 700 Mhz. 
  • Windows Me. o bien  Windows  XP.

Instalar 

Para Instalar el Editor Simulador de Robots Oientado a Objetos 2002 en Windows:

1. Doble clic en el archivo "Instalar .exe" que se encuentra en la carpeta:

        programa/insData/Windows/VM/Instalar.exe  para Maquina Virtual de Java.

o bien 

        programa/insData/Windows/NoVM/Instalar.exe si su sistema no cuenta con Maquina Virtual de Java. 

 

2. Siga las instrucciones del Instalador.

Adicionalmente 

    El CD de Instalación contiene aplicaciones complementarias, a saber:

·        Para funcionar correctamente el programa requiere que la computadora en donde se instalará cuente con la version JDK 1.3 o mayor.  Para instalar el JDK 1.3 para Windows en su computadora haga doble clic en el archivo:

        software/j2sdk-1_3_0_02-win.exe

·        Si necesita actualizar los controladores DirectX a la version 8.0, el archivo correspondiente se encuentra en:

        software/DX80spa.exe

·        Finalmente si su sistema requiere la instalación del API 3D de Java, el archivo compatible con  DirectX se encuentra en :

        software/java3d-1_2_1-win32-directx-sdk.exe
